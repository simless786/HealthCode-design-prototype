# HealthCode-design-prototype
파편화된 건강 데이터를 통합한 올인원 헬스 케어 앱 '건강 코드'의 UI/UX 프로토타입. 순수 프로토타입 UI용.
아래 프로젝트 중 순수 UI 프로토타입 생성용입니다.

[건강 코드 (Health Code)]
본 프로젝트는 2026년 컴퓨터소프트웨어과 캡스톤 디자인 과정으로 개발 중인 통합 건강관리 서비스입니다. 기존에 식단, 운동, 복약 등으로 분산되어 있던 건강 관리 기능을 하나의 플랫폼으로 구축하여 자기관리의 효율성을 극대화하는 것을 목표로 합니다.

[주요 기능 (Key Features)]
* 메인 올인원 대시보드: 일일 활동량(칼로리, 걸음 수, 활동 시간) 및 바이탈 데이터(심박수, 수분) 통합 시각화
* 정시 복약 알림 센터: 식약처 API 연동 및 정시 푸시 알림, 달력 기반 복약 이력 관리
* AI 음식 자동 인식: 카메라 촬영 시 머신러닝 알고리즘 기반 음식명 및 칼로리 자동 분석·입력
* 종합 건강 리포트: Java 연산 기반 종합 점수 산출 및 동일 연령대(20대) 대조군 비교 시각화 그래프 제공
* 건강검진 혜택 알리미: 공공데이터포털 API 연동을 통한 연령/성별/특수 자격(국가보훈자 등) 맞춤형 무료 검진 정보 안내
* 친구 랭킹 시스템: 조원 및 친구 간의 주간 종합 건강 점수 순위 경쟁 소셜 기능 (정보 공개 범위 제어 포함)

[개발 환경 및 기술 스택 (Tech Stack)]
* Frontend: HTML5, CSS3 (Glassmorphism UI), JavaScript (ES6+)
* Mobile OS Environment: Android 13.0 (API Level 33) 이상 지원 예정
* IDE: Android Studio, VS Code
* Backend & DB: Firebase (Authentication, Firestore, FCM, Crashlytics), Python 서버

(Team Members)
* 양서윤: 파이썬 서브, 데이터 서브 (데이터 신뢰성 검증 및 문서화), UI 서브
* 본인 외의 다른 팀원들은 실명 보호로 여기서는 작성하지 않음.
